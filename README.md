# MobileHomeWork

Бикмухаметов МПБ-301-О-01

Краткое описание: Android приложение для управления задачами (To-Do List), созданное с помощью Jetpack Compose. Позволяет пользователям создавать, редактировать, удалять и отслеживать свои задачи с удобным интерфейсом и функционалом поиска, фильтрации и сортировки.

## Стек

- **Язык и фреймворк**: Kotlin, Jetpack Compose, Android SDK
- **Ключевые библиотеки**:
  - Jetpack Compose (UI)
  - Material Design 3 (дизайн система)
  - Gson (сериализация данных)
  - SharedPreferences (локальное хранилище)
- **Сервисы и интеграции**: Локальное хранение данных через SharedPreferences

## Быстрый старт

### Требования

- Android Studio Hedgehog или новее
- Min SDK: 24
- Target SDK: 36
- Kotlin 2.0.21
- Gradle 8.12.3

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/KillerQuen381/MobileHOmeWork.git
```

2. Откройте проект в Android Studio

3. Дождитесь синхронизации Gradle

4. Запустите приложение на эмуляторе или устройстве

### Использование

1. **Добавление задачи**: Введите текст в поле "Новая задача" и нажмите кнопку добавления
2. **Редактирование**: Нажмите на иконку редактирования рядом с задачей
3. **Выполнение**: Отметьте чекбокс для отметки задачи как выполненной
4. **Удаление**: Нажмите на иконку корзины для удаления задачи
5. **Поиск**: Используйте поле поиска для быстрого нахождения задач
6. **Фильтрация**: Выберите фильтр (Все/Активные/Выполненные) для просмотра нужных задач
7. **Сортировка**: Нажмите на иконку сортировки в AppBar для выбора типа упорядочивания задач
